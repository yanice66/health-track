## Project Summary
A health-focused web application featuring a BMI calculator and coaching platform. The UI is designed to be clean, modern, and accessible, with a focus on wellness and health tracking.

## Tech Stack
- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS
- Icons: Lucide React
- Components: Radix UI (Tabs)
- Animations: Framer Motion

## Architecture
- `src/components/health-track/`: Contains modular UI components specific to the HealthTrack application.
  - `Navbar.tsx`: Global navigation.
  - `BMICalculator.tsx`: Interactive calculator with metric/imperial support.
  - `HeroCard.tsx`: Promotional card with background image and call-to-action.
- `src/app/page.tsx`: Main entry point with responsive grid layout.

## User Preferences
- Preferred vibrant teal color: `#14F1C9`.
- Clean, high-quality photography for health context.
- Minimalist and modern typography.

## Project Guidelines
- Use Lucide icons for visual consistency.
- Ensure responsive design for mobile and desktop.
- Maintain a high-contrast ratio for readability.

## Common Patterns
- Teal accents for primary actions and highlights.
- Rounded corners (3xl) for cards and interactive elements.
- Soft shadows and subtle border-zinc-100 for depth.
